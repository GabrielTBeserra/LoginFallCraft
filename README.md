# LoginFallCraft

### Configuracaoes
   - Banco de dados disponiveis (MYSQL/SQLITE)

### Comandos
   - /register (senha) (senha)
   - /unregister (nome)
   - /login (senha)
   - /changepassword (senha)
   - /loginfallcraft reload

### Permissoes
   - `loginfall.unregister`
   - `loginfall.reload`
   
### Instalacao
    Colocar o arquivo na pasta de plugins do BungeeCord e do Spigot
